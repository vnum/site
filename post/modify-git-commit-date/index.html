Modify Git Commit Date

当需要修改git commit时间，可以通过以下方法。

  1. 修改本地 commit 时间
    git commit --amend --date="2021-08-02T20:38:42+0800"
  2. 修改之前某次 commit 时间，先git log 获取 commit id
    git commit --amend --date="2021-08-02T20:38:42+0800" -C commit-id
